# File Map

Подробная карта файлов архива **Norton DeskTop for Windows 2.0**.

## Основные утилиты

| Файл | Размер | Назначение |
| --- | ---: | --- |
| `Norton DeskTop for Windows 2.0/NAV.EXE` | 145 817 B | Norton AntiVirus / основная NAV-утилита |
| `Norton DeskTop for Windows 2.0/NDD.EXE` | 137 817 B | Norton Disk Doctor |
| `Norton DeskTop for Windows 2.0/SPEEDISK.EXE` | 104 969 B | оптимизация/дефрагментация диска |
| `Norton DeskTop for Windows 2.0/UNERASE.EXE` | 139 785 B | восстановление удаленных файлов |
| `Norton DeskTop for Windows 2.0/UNFORMAT.EXE` | 79 913 B | восстановление после форматирования |
| `Norton DeskTop for Windows 2.0/NPACK.EXE` | 33 928 B | служебная Norton-утилита |

## Системные и Windows-компоненты

| Файл | Размер | Назначение |
| --- | ---: | --- |
| `Norton DeskTop for Windows 2.0/NAV&.SYS` | 18 851 B | системный компонент NAV |
| `Norton DeskTop for Windows 2.0/NAV_.SYS` | 55 598 B | системный компонент NAV |
| `Norton DeskTop for Windows 2.0/NAVDLL.DL$` | 4 945 B | сжатый Windows DLL-компонент |
| `Norton DeskTop for Windows 2.0/NAVPOPUP.EX$` | 2 632 B | сжатый Windows executable-компонент |
| `Norton DeskTop for Windows 2.0/VNAVD.38$` | 1 292 B | сжатый Windows 386/VxD-компонент |
| `Norton DeskTop for Windows 2.0/NORTON.INI` | 972 B | служебный INI/конфигурационные данные |

## Примечания

- Файлы с расширением `.$` выглядят как сжатые установочные компоненты старого Windows/DOS-комплекта.
- `NORTON.INI` содержит бинарные/служебные данные, поэтому его лучше не редактировать вручную.
- Для проверки целостности используйте [`CHECKSUMS.txt`](CHECKSUMS.txt).
